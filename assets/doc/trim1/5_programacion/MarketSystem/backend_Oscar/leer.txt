Muchachos deben crear los archivos ya que estos no van en el push a github por seguridad:

.env
.prod.env
.stag.env

dentro de estos archivos las siguientes variables de ambiente:

DATABASE_NAME=my_db
DATABASE_PORT=1000
API_KEY='4321'
PORT=300

MYSQL_DATABASE=Market_System
MYSQL_USER=root
MYSQL_ROOT_PASSWORD=123456
MYSQL_PORT=3306
MYSQL_HOST=localhost

______________________

En caso de que no funcionen sin docker como contenedor de estas (aunque creo que con xamp igual sirve)

los comandos despues de descargar docker:

docker-compose up -d mysql    // para correr el contenedor de mysql
docker-compose up -d phpmyadmin  // para correr el contenedor de la interfaz grafica de php, localhost//8080

docker-compose ps // para verificar que los dos esten corriendo

docker-compose down // para bajar o detener el contenedor de docker
